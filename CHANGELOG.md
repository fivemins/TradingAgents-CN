# Changelog

All notable changes to this public fork will be documented in this file.

## v0.1.0 - 2026-03-24

First public release of `TradingAgents-CN`.

### Added

- Chinese dashboard experience for local single-user workflows
- A-share single-stock analysis flow
- Overnight scanner, next-open validation, and 60-trading-day historical review
- Structured decision outputs and structured factor snapshots
- Windows-friendly startup scripts and deployment guides
- QVeris supplemental live data support for A-share overnight scanning

### Changed

- Reworked the project for A-share research and overnight candidate workflows
- Improved Chinese-first report generation and dashboard presentation
- Replaced legacy technical-indicator runtime dependencies with an internal indicator pipeline
- Hardened repository hygiene for public distribution

### Notes

- This release is intended as the first public baseline for collaboration and deployment.
- Live data quality still depends on upstream providers and network availability.
