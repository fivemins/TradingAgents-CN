repository page, including the right-side description, topics, and optional
homepage links.

If you want to set a homepage URL in the GitHub About panel, use one of these:

- project demo or deployment guide URL
- documentation landing page URL
- your public project page

If you do not have a dedicated homepage yet, leave it empty.
